# CTDesk

Advanced Customer Support Ticketing System built with Python Flask and SQLite.

## Features

- Login system
- Admin and agent roles
- Ticket creation, editing and assignment
- Ticket priorities and statuses
- SLA deadline tracking
- Customer database
- Public replies and internal notes
- Activity history
- Dashboard statistics
- Search and filters
- Knowledge base
- CSV export
- Light/dark mode

## How to run

```bash
pip install -r requirements.txt
python database.py
python app.py
```

Then open:

```text
http://127.0.0.1:5000
```
